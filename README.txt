Игра, написанная для итогового проекта курса "Flask: введение".
Смысл в том, чтобы создать поле произвольных размеров и собрать всё золото, представленное в виде желтых клеток.
Также есть невидимые вражеские клетки, при наступлении на которые теряется жизнь.

Недостатком является то, что мир создается при отправлении форм в index.html, поэтому при обновлении страницы возникается ошибка.
